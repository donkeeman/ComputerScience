# 운영체제 (Operating System)
- **컴퓨터를 작동**시키고 **운영을 관리**하여 응용 프로그램이 효율적으로 실행될 수 있는 환경을 제공하는 시스템 소프트웨어
- 응용 프로그램과 하드웨어 사이에서 응용 프로그램들이 하드웨어의 **자원**을 사용할 수 있도록 **할당 및 관리**
- **하드웨어 추상화** 플랫폼 제공
- 프로그램의 오류나 잘못된 자원 사용을 감시
- **가상 머신**에서 실행할 수도 있음

## 싱글태스킹 운영체제
- **한 번에 하나의 프로그램**만 실행할 수 있는 운영체제
- 실행 중인 프로그램의 작업이 끝나야 다른 프로그램을 실행할 수 있음 (**블로킹**)

## 멀티태스킹 운영체제
- **한 번에 여러 프로그램**을 실행할 수 있는 운영체제
- 실제로 동시에 실행되는 것이 아니라 운영체제가 CPU 사용권을 적절하게 분배해서 (**프로세스 스케줄링**) 여러 프로그램이 번갈아가면서 수행
- 예전에는 프로세스 단위인 멀티프로세싱이었는데 하드웨어 자원의 부담이 커서 스레드 단위의 **멀티스레딩**이 도입
### 선점형 멀티태스킹
- **운영체제가 CPU의 점유를 제어**하므로 특정 프로세스가 CPU를 독점할 수 없음
- 적절한 시간 단위(퀀텀)마다 프로세스의 **선점**이 일어남
- 프로세스 실행 중 오류가 발생하면 운영체제가 그 프로세스를 강제 종료시키고 모든 자원 반환
### 협동형 멀티태스킹 (비선점형 멀티태스킹)
- **프로세스가 CPU의 점유를 제어**하므로 특정 프로세스가 CPU를 독점하기도 함
- 프로세스가 유휴 상태가 되면 자발적으로 제어권을 **양보**
- 프로세스가 자발적으로 실행을 종료하지 않는 한 강제로 프로세스를 종료시킬 수 없음

## 단일 사용자 운영체제
- **한 번에 한 명의 사용자**만 사용할 수 있는 운영체제
- 다중 작업은 허용
## 다중 사용자 운영체제
- **한 번에 여러 명의 사용자**가 사용할 수 있는 운영체제

## 디스크 운영체제 (DOS)
- 플로피 디스크, 하드 디스크 드라이브, 광 디스크 등의 **저장 장치**를 사용할 수 있는 운영체제
- 파일 정리, 읽기, 쓰기를 위한 **파일 시스템 제공**
- 현재는 적용되지 않음

## 분산 운영체제
- **네트워크**로 연결된 각각의 컴퓨터들을 관리하는 운영체제
- 각각의 컴퓨터들을 **하나의 컴퓨터로 인식**
- 연산을 분산시켜 한 대 이상의 컴퓨터에서 수행 (**협업**)
- 자원 공유로 인해 속도, 용량 효율 상승
- 보안, 비용 문제가 있을 수 있음

## 판형 운영체제
- 하나의 운영체제를 **쪼개어** 여러 개로 사용하는 운영체제
- 사용자에게 쪼갠 운영체제를 **임대**
- 가상화, 클라우드 등 **대형 서버 웨어하우스 환경**에서 사용

## 임베디드 운영체제
- 스마트폰, 내비게이션, 라우터 등 **임베디드 장치**를 제어하는 운영체제
- 크기가 작고 효율적으로 설계됨

## 실시간 운영체제
- 실시간 응용 프로그램을 위해 **특정 짧은 시간** 내에 이벤트나 데이터를 처리하는 운영체제
- 멀티태스킹일 경우 특수 스케줄링 알고리즘 사용

## 라이브러리 운영체제
- 운영체제가 제공하는 서비스들이 **라이브러리 형태**로 제공되는 것

## 대표적인 운영체제

### Windows
- 마이크로소프트 사에서 개발한 운영체제
- 사유 소프트웨어
- **개인용 컴퓨터**에 주로 사용
- **호환**되는 응용 프로그램이 많음

### Mac OS
- 애플 사에서 개발한 **매킨토시 용** GUI 운영체제
- 사유 소프트웨어
- 최신 버전인 macOS는 유닉스 계열 운영체제
- 매킨토시 및 인텔 맥과만 호환

### Unix
- 벨 연구소에서 개발한 운영체제
- 사유 소프트웨어
- 멀티태스킹과 다중 사용자를 지원하도록 설계됨
- 여러 회사들과 비영리 단체들이 **유닉스 커널을 활용**하여 **다양한 운영체제**를 개발

### Linux
- 리누스 토르발스가 개발한 **유닉스 계열**의 운영체제
- **자유 소프트웨어**
- **오픈 소스**
- 개인용 컴퓨터, 서버, 임베디드 장치, 모바일 등 다양한 플랫폼에서 사용되지만 **서버**에서 가장 많이 사용
- TOP 500 슈퍼컴퓨터에 쓰이는 유일한 운영체제
- **다양한 리눅스 배포판**이 있음

### GNU
- 리처드 스톨먼이 개발한 **유닉스 계열**의 운영체제
- 유닉스 코드는 포함하지 않음
- **자유 소프트웨어**
- **리눅스 커널을 결합**한 형태가 가장 많이 사용됨
